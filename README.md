# 🎵 Music App

A simple Python-based music playback application built with **pygame**.  
This project demonstrates how to initialize the mixer, load audio files, and play them using minimal code.

---

## 🚀 Features
- Initialize the `pygame.mixer` audio engine  
- Load and play music from a specified file path  
- Lightweight and easy to extend with pause, resume, and stop controls  

---

## 🧩 Requirements
- Python 3.11 or later  
- `pygame` 2.6.1 or later
